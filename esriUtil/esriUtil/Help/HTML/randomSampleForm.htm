Create Random Sample

This form is used to create a point feature class that represents a random sample of raster cell values. The point locations in the output sample file will have a field named Value that identifies the raster value at the location of that sample point.

Raster to Sample dropdown and open button: Allows users to select the raster layer from which cells will be randomly selected.
Output Workspace text box and open button: Allows users to select the location where the sample file will be stored.
Name of New Sample File textbox: Allows user to specify the name of the new sample file.
Number of Samples numeric dropdown: Allows the user to specify the total number of samples.
Create Sample button: Creates the new sample point feature class in the specified location.

Output

A sample point feature class.
