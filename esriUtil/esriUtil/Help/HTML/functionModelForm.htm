Function Modeling

This form is used to perform Function Modeling. Using this form users can dynamically create, save, load, and add to map the outputs of function models. Most tools described within the Raster Analysis menu are available when using this form. This form allows users to stack multiple functions together as one model and save that model to a .fds file. This file can then be loaded at anytime back into this form or into the map using the Open Function Model form.

Function Groups listbox: Allows users to open a function modeling form. After executing the selected function, the outputs are added to the Function Datasets data grid.
Function datasets data grid: Lists the outputs of the function model processes that have been performed. To remove a function model from the data grid highlight the model and click the - button. To clear the data grid click on the ! button.
Load button: Allows users to load a previously saved function model. Function models are saved to a functionDataset folder located within a workspace directory specified at the time of saving.
Save button: Saves all checked function model to disk. Function models are saved to a functionDataset folder located within the specified workspace.
Close / Add to Map button: Closes this form and adds all checked models to the map.
