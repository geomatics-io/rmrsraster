Add Raster Surface to Map

This form is used to add selected raster datasets to an active map.

Select Raster drop down: Allows users to select which raster surfaces they want to load into the active map's TOC. Selecting a raster will add it to the selected raster listbox
% button: Selects all web services raster surfaces within a given database
- button: Removes a selected raster surfaces from the list of selected raster surfaces
