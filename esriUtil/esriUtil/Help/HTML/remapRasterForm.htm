Remap Raster

This form is used to remap cell values of a specified raster. The outputs from the analysis consist of a raster dataset with cell values based on the remapped table. Unspecified ranges of values will default to original cell values.

In Raster dropdown and open button: Allows users to select the raster that will be remapped. The dropdown contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Remap Table data grid: Allows users to select a range of values that will be remapped. >= Min and < Max define a range of values that will be remapped to what is specified in the value field of the remap table. To add a range of values use the + button. To remove a range of values use the - button. To remove all remap ranges use the ! button.
Out Raster Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
