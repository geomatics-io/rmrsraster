Create GLCM Form

This form is used to perform a GLCM analysis.

Raster dropdown and open button: Allows users to select which raster datasets will be used in the GLCM analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Out Name text box: Allows users to specify the name of the newly created mosaic dataset.
Window Type dropdown: Allows users to specify GLCM analysis window type. Window types include Rectangle and Circle. If a circle window type is chosen, then columns and rows will be replaced with radius.
Columns, Rows, and Radius numeric up down: Allows users to specify the sized of the analysis window (Cells).
Direction dropdown: Allows user to specify the direction of the GLCM comparison.
Types dropdown and listbox: Allows users to select which types of GLCM metrics will be calculated. Each type chosen will be represented as a band within a multiband raster. The order of bands correspond to the order of selected metrics.To remove a given metrics select it within the listbox and click the - button. To clear the listbox click the ! button.
