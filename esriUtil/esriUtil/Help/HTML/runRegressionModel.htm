Run Regression

This form is used to create new raster surfaces using the outputs from the Model Regression tool.

Training Dataset and open button: Allows users to select the training feature class that represents the sample locations. The workspace of this feature class should contain the outputs of the model building stage.
Rasters dropdown, open button and list box: Allows users to select the raster surfaces that were be used as independent variables to build the classification model. Note that raster surfaces should be in  the same order as specified in the model output (ascending order). To view that order click the Order of Raster Button.
- button: Allows users to remove the currently selected field from the Independent Fields list box.
% button: Allows users to add all fields from the training dataset.
! button: Allows users to remove all fields within the Independent Fields list box.
Order of Raster button: When clicked will open a window that displays the name and order of independent variables.
Create button: Executes the Regression model. Currently this process creates a conventional raster dataset. Future releases will create function datasets.

Output
All outputs are located within the workspace of the training feature class and are named REG. REG raster dataset cell values depict the mean estimate of the regression model given the independent datasets.
