Summarizing Biomass

This form is used to summarize biomass estimates (Jenkins et. al., 2004) by plot/subplot given a series of sample locations and a FIA access database.

Sample locations dropdown and open button: Allows users to select the point feature class that represents the sample locations.
Plot link field dropdown: The field used to specify the link between the plots and the FIA database.
SubPlot link field dropdown: The field used to specify the link between the subplots and the FIA database.
FIA database textbox and open button: Allows users to select the downloaded FIA database.
AGB types check list box: Allows users to select which values will be summarized. For each species found within all the samples, checked items will be added to the sample feature class as a field, named according to the checked item followed by a "_" and FIA species coded value.
Execute button: Executes the summarization routine. Summarized values for each item tree species combo are populated based on the species found at each plot/subplot and represent mean per acre values.

Output

A series of fields and summarized FIA values based on user inputs.
