Model PLR

This form is used to build Logistic Regression and Polytomous Logistic Regression Models. Currently this tool requires SAS to generate the PLR model. After the model has been created it can be used along with the Run PLR Classification tool to create a MLC and Probabilistic raster datasets.

Training Dataset and open button: Allows users to select the training feature class that represents the sample locations.
Dependent Field dropdown: The field used to identify the response variable.
Weight Field dropdown: The field used to specify the weight of each sample.
Stepwise Selection checkbox and % Alpha Enter and Stay: Allows users to select if they want to use stepwise selection to select statistically valid parameters (within the bounds of the Enter and Stay). If the check box is not selected then all parameters will be used in the model.
Independent Fields dropdown and  list box: Allows users to select which independent fields will  potentially be added to the model. After selecting a given field it will be added to the Independent Fields list box.
- button: Allows users to remove the currently selected field from the Independent Fields list box.
% button: Allows users to add all fields from the training dataset.
! button: Allows users to remove all fields within the Independent Fields list box.
Validation checkbox: When checked will perform an independent validation of the data. 70% of the data will be used to create the model and the remaining 30% of the data will be used to validate the model.
View Results checkbox: When checked SAS results will be shown after the process has completed.
Edit SAS checkbox: When checked the prc.sas file will be opened for editing.
Execute button: Executes the PLR routine. After clicking the Execute button a batch SAS process will open, running the prc.sas code.

Output
All outputs are located within a the workspace of the training feature class and are stored within a directory called SASOUTPUTS\GENERILIZED_LOGISTIC or SASOUTPUTS\LOGISTIC depending on the type of analysis. Outputs include:

  1. A series of graphical outputs depicting the relationship between response and explanatory variables.
  2. A .csv file of the input data named after the training data feature class.
  3. A .csv file of the model parameter estimates. This file is used by the Run PLR Classification tool to build series of raster datasets that depict the outputs of the model.
  4. A SAS log file named prc.log. This file provides all the log information produced from running the SAS procedure file.
  5. A SAS results file named prc.lst. This file provides all the results produced from running the SAS procedure file.
  6. A SAS procedure file named prc.sas. This file contains the SAS code that is submitted to SAS in the batch process. You can open this file and modify the SAS procedure to produce different outputs. Please note that outputs from export statements are used by other tools and these tools may not work properly if default names and locations are changed.
