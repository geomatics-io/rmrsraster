Arithmetic Analysis

This form is used to perform arithmetic analyses between 2 raster datasets or a raster dataset and a constant. Types of analyses include; plus, minus, multiply, divide, power, and modules division. If both raster datasets are multiband raster datasets with an equal number of raster bands than the tool will perform arithmetic analysis between each corresponding band of the 2 raster datasets. If one raster is a multiband raster and the other is a constant or raster with fewer bands, then the first band or constant will be added to the raster with the fewest bands to make both raster datasets have the same total number of bands. There must be at least one raster dataset specified for the tool to work.

Input Raster 1 or number dropdown and open button: Allows users to select which raster or constant will represent the first raster in the analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To specify a constant the user must manually type in number.
Input Raster 2 or number dropdown and open button: Allows users to select which raster or constant will represent the 2nd raster in the analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To specify a constant the user must manually type in number.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
