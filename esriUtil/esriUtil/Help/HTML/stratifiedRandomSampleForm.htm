Create Stratified Random Sample

This form is used to create a point feature class that represents a stratified random sample of raster cell locations. Point locations in the output sample file will have a field named Value, Category, and Weight that identifies the raster value at that sample location, an empty text field, and the weight of that sample given the size of the strata, respectively.

Raster to Sample dropdown and open button: Allows users to select the raster layer from which cells will be randomly selected.
Output Workspace text box and open button: Allows users to select the location of the output sample file.
Name of New Sample File textbox: Allows user to specify the name of the new sample file.
Samples per Strata numeric dropdown: Allows the user to specify the number of samples within each strata.
Create Sample button: Creates the new stratified sample point feature class in the specified location.

Output

A point feature class.
