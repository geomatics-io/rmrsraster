Sample Raster

This form is used to sample a specified raster. Raster values will be added to the attribute table of the specified feature class. Field names will include the name of the raster sampled followed by the band number that was sampled. Sampled values correspond to the value field of the Raster.

Sample locations dropdown and open button: Allows users to select the point feature class that represents the sample locations.
Raster to Sample dropdown and open button: Allows users to select the raster that will be sampled. All bands within the specified raster will be sampled.
Sample button: Samples the specified Raster.

Output

A series of new fields that depict the GLCM metric values of the raster surface bands at the location of each sample.
