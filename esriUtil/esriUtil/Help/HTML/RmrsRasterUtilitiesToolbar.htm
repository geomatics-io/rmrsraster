Toolbar Overview

RMRS Raster Utility toolbar is a series of GIS tools built around Environmental Systems Research Institute (ESRI) ArcGIS software. RMRS Raster Utility was designed using an Object-Oriented architecture, making it easy to modify and adapt to new applications. Main features of the RMRS Raster Utility toolbar include:

  • Simplified data acquisition from GIS web Services
  • Significantly improved reading, writing, and processing of raster data
  • Sampling design, creation and spatial extraction
  • Easy access to standard raster analysis techniques
  • Gray level co-occurrence matrix analysis
  • Moving windows landscape metric analysis
  • Simplified statistical modeling
  • Function modeling

The toolbar is organized into 4 main buttons and 5 main menus. Buttons include:

  •  Save raster
  •  Open Function Model
  •  Delete Function Model
  •  Calculate Raster Statistics

Menus include:

  • Web Services
  • Sample
  • Raster Analysis
  • Statistical Modeling
  • Help
