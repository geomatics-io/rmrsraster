GLCM overview

Gray Level Co-Occurrence Matrix (GLCM) was first described by Haralick in the early 1970's and has since been used in numerous disciplines to quantify pattern within an image given a specified analysis window. The basic idea behind GLCM is to quantify differences between neighboring pixel values (vertically or horizontally) within a specified window of interest (figure 1). Typically, GLCMs are good at quantifying and identifying different aspects of edge and or homogenous area for a given analysis window within an image. The different GLCM metrics can be grouped into 4 categories that represent Contrast, Statistical, Orderliness, or Extreme value relationships (figure 2). The outputs of GLCM can be used as predictor variables in a model and can be easily viewed as a 3 band color composite (figure 3).

 

Figure 1. GLCM values are a summary statistic of all neighbor pixel relationships in a given direction (vertical or horizontal). Like focal analyses GLCM summary values are applied to the center cell of the analysis window. To calculate GLCM values, neighboring value combination within a given window analysis area (bottom left analysis window) are counted and recorded as counts. Counts are then used to generate a probability of occurrence (Pij) for each cell within the GLCM matrix (right, I, J matrix). Finally, probability of occurrence are transformed using multiple equations that highlight unique aspects of neighboring pixel differences. 


Figure 2. GLCM equations. All equation transform neighbor pixel combination probabilities and then summarize those transformations to produce a metric value using one of the above equations. GLCM equations can be grouped into one of four categories that describe the intent of the transformation; Contrast (blue text), Statistical (green text), Orderliness (red text), and Extreme (purple text).

                                         


Figure 3. Three band color composite of horizontal Contrast, Orderliness, Statistical, and Extreme value GLCM values using a 3*3 analysis window and the first band of color NAIP imagery. The black line in all 4 images serves as a reference. Note how different patterns emerge given the different GLCM metrics.
