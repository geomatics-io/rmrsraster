Run PLR Classification

This form is used to create new raster surfaces using the ouputs from the Model PLR tool.

Training Dataset and open button: Allows users to select the training feature class that represents the sample locations. The workspace of this feature class should contain the outputs of the model building stage.
Rasters dropdown, open button and list box: Allows users to select which raster surfaces were be used as independent variables to build the classification model. Note that raster surfaces should be in the same order as specified in the model output (ascending order). To view that order click the Order of Raster Button.
- button: Allows users to remove the currently selected field from the Independent Fields listbox.
% button: Allows users to add all fields from the training dataset.
! button: Allows users to remove all fields within the Independent Fields list box.
Order of Raster button: When clicked will open a window that displays the name and order of independent variables.
Create button: Executes the PLR model. Currently this process creates a conventional raster dataset. Future releases will create function datasets.

Output
All outputs are located within the workspace of the training feature class and are named MLC and PROB. MLC layer defines the most probable class given the modeled probabilities. The PROB raster depicts the probability of each class given the independent variables. Raster band order correspond to the ordering of the PLR categories.
