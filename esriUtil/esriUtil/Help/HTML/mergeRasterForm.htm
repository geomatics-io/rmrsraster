Merge Raster Form

This form is used to merge raster datasets together. Outputs from this tool are stored as a mosaic dataset within the users local temp folder.

Input Raster dropdown and open button: Allows users to select which raster datasets will be used in the merge process. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To remove a raster dataset from the listbox select it from the listbox and click the - button. To clear the input listbox click the ! button.
Merge Type dropdown: Allows users to specify how overlaps will be handled. Merge types include: First, Last, Min, Max, Mean, and Blend
Out Name text box: Allows users to specify the name of the newly created raster dataset.
Execute button: performs the merge operation. Note, not all images will show up in the display if you are zoomed out past the number of records to display. To see the those values zoom in to a larger scale.
