Convert Pixel Type

This form is used to convert the pixel type of a raster datasets to a different pixel type. Function modeling stores all representation of raster datasets in 64 bit double precision. Before saving a raster dataset to a conventional format, users may want to convert the pixel type to a small bit size. Note, "no data" values are converted to min or max values depending on whether the pixel type is signed or unsigned (link to supported pixel types)

Input Raster dropdown and open button: Allows users to select the raster layer to convert to a different pixel type. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Pixel Type dropdown: Allows users to select the new pixel type of the output raster. Note, output raster datasets no data value will also be converted to the min or max of the specified pixel type.
Out Name text box: Allows users to specify the name of the newly created raster dataset.
Execute button: performs the convert pixel type operation.
