Conditional Analysis

This form is used to perform conditional if then analyses given a conditional raster (cell values of 1 or 0) and two raster datasets or constants. The outputs from the analysis consist of a raster dataset with each cell having a value equal to the constant or raster of the True or False condition.

Conditional Raster dropdown and open button: Allows users to select which raster will be used as the conditional raster. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. Conditional raster dataset cell values are assumed to be 0 or 1
Condition True (1) Raster or value dropdown and open button: Allows users to specify a constant or select which raster will be used for the true case of the conditional raster (values of 1). The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Condition False (0) Raster or value dropdown and open button: Allows users to specify a constant or select which raster will be used for the false case of the conditional raster (values of 0). The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
