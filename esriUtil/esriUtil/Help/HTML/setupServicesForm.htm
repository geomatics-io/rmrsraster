Service Setup

This form is used to setup a web service database and identify which services will be queried and which layers from that service will be downloaded to the output database (DB). To select a connection and service use the connection and service dropdown controls. Once a connection and service has been specified, a list of available layers will appear in the check listbox. All checked layers for all connections and services will be downloaded when the service download button is clicked. When you are finished selecting layers to download simply close this form and use the service download button. Once downloaded, these layers can be added to a map using the buttons. Note: to use this tool you must be connected to the internet.

Change Output DB button: Allows users to select which database will be used to store web service information
+ button: Open the Add Service Connection form that allows user to add another ArcGIS Server Connection (connection to a web service)
- button: Removes a selected ArcGIS Server Connection from the current output DB
ArcGIS Server Connection dropdown: The currently active web service connection
Service Name dropdown: The currently active service
Download Data for Checked Layers checked list box: Check layers that you want to download from a given service for a given ArcGIS connection
