Open Function Model

This form is used to to add the outputs of a function model to the active view.

File name dropdown: Allows users to select a function model. Function models are stored in a functionDataset directory located within a given workspace and have a .fds extension.
Files of type: Allows users to select a function models with a .fds extension.
Open button: Opens the selected function model and adds the output (Raster) to the active view.
Cancel button: Close the open function model dialog.
