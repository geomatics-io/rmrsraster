Raster Analysis Menu

The Raster Analysis Menu contains 19 buttons that can be used to perform a wide arrange of spatial analyses. Outputs from each analysis are stored as Function Datasets with a pixel depth of 64 bit double precision. Sometimes outputs may not display correctly due to the way statistics are estimated. If outputs do not display correctly, try calculating statistics using the Calculate Statistics tool. Using the Function Modeling button and form users can save Function Models that can be loaded into ArcMap at a later time. Alternatively, users can save a function dataset as a conventional raster dataset using the Save Raster form. Prior to saving to a conventional raster dataset users may want to convert the pixel depth to something smaller than 64 bit double precision. To do this please use the Convert Pixel Type tool. The merge and mosaic tools create mosaic datasets that currently do not work with focal and local type analyses. As a workaround please perform focal/local analyses before merging/mosaicing. Alternatively users can save the outputs from the merge/mosaic tools to a conventional dataset before using focal/local analyses.
