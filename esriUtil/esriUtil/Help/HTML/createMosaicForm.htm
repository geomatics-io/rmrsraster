Mosaic Raster Form

This form is used to mosaic raster datasets together. Outputs from this tool are stored as a mosaic dataset within the user specified workspace. For more information about mosaic methods and types please visit ESRI's online help (link).

Input Raster dropdown and open button: Allows users to select the raster datasets that will be used in the mosaic process. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk. To remove a raster dataset from the listbox select it from the listbox and click the - button. To clear the input listbox click the ! button.
Method dropdown: Allows users to specify the mosaic method. Mosaic methods include: None, Center, Nadir, Viewpoint, Attribute, Lockraster, Northwest, and Seamline.
Type dropdown: Allows users to specify how overlaps will be handled. Merge types include: First, Last, Min, Max, Mean, and Blend
Out Workspace dropdown and open button: Allows users to select the workspace used to store the mosaic dataset
Out Mosaic Name text box: Allows users to specify the name of the newly created mosaic dataset.
Footprint check box: When checked will create a footprint of each image and store it as part of the mosaic dataset.
Boundary check box: When checked will create a Boundary of all images and store it as part of the mosaic dataset.
Overview check box: When checked will create an image overview. This will increase the time it take to run the mosaic.
Seamlines check box: When checked will create a seamline layer and store it as part of the mosaic dataset.
Execute button: Performs the mosaic operation. Note, not all images will show up in the display if you are zoomed out past the number of records to display. To see the those values zoom in to a larger scale.
