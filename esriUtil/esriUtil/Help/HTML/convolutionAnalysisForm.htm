Convolution Analysis

This form is used to perform convolution analyses given a raster datasets and specified convolution kernel. Convolution analyses use the same neighboring window concept as focal analyses but allow the user to specify the values of the kernel. Every band within the raster datasets will be transformed based on the analysis specifications. The outputs from the analysis consist of a raster dataset with transformed cell values based on neighboring cell values and the convolution kernel.

Input Raster dropdown and open button: Allows users to select which raster will be used as an input to the convolution analysis. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Kernel size and values: Allows users to define the convolution kernel and it values.
Columns: Allows users to select the number of columns (cells) for a given kernel.
Rows: Allows users to select the number of rows (cells) for a given kernel.
Load button: Allows users to load a previously saved kernels.
Save button: Allows users to save a defined kernel.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
