Math Transformation

This form is used to perform mathematical transformations of a given raster dataset. Types of transformations include; log, natural log, exponentiation, ^10, sine, cosine, tangent, arcsine, arccosine, arctangent, and radians. Every band within the input raster will be transformed given the specified transformation. Note, sine, cosine, tangent, arcsine, arccosine, and arctangent assume cell values are in radians. If cell values are in degrees first convert degrees to radians using the radians transformation.

Input Raster dropdown and open button: Allows users to select which raster or constant will be used in the transformation. The drop down contains any raster in the TOC and the open button allows the user to select any existing raster stored on disk.
Transformation dropdown: Allows users to select which type of transformation will be performed.
Out Name text box: Allows users to specify the name of the newly created function raster dataset.
Execute button: Executes the analysis. Outputs are stored as function datasets and can be saved as a conventional dataset using the Save Raster form.
